# skulMate Character Assets

This directory contains character images for skulMate learning companions.

## Character Structure

### Age Groups

1. **Elementary (5-10 years)**
   - `elementary_male.png` - Skully (Male character)
   - `elementary_female.png` - Skylar (Female character)

2. **Middle School (11-14 years)**
   - `middle_male.png` - Max (Male character)
   - `middle_female.png` - Maya (Female character)

3. **High School (15-18 years)**
   - `high_male.png` - Alex (Male character)
   - `high_female.png` - Aria (Female character)

## Image Specifications

- **Format**: PNG (with transparent background recommended)
- **Dimensions**: 512x512px or higher (square format)
- **Style**: Friendly, approachable, age-appropriate
- **Background**: Transparent or solid color
- **Quality**: High resolution for crisp display

## Character Design Guidelines

### Elementary Characters (5-10 years)
- Playful and energetic
- Bright, cheerful colors
- Simple, friendly expressions
- School-themed accessories (backpack, books, etc.)

### Middle School Characters (11-14 years)
- Confident and determined
- Modern, relatable appearance
- Balanced between playful and mature
- Learning-focused accessories

### High School Characters (15-18 years)
- Focused and ambitious
- More mature appearance
- Professional yet approachable
- Achievement-oriented accessories

## Usage

Characters are automatically loaded based on user selection. The system will:
1. Display character selection screen on first use
2. Store user's choice in database and local preferences
3. Show character in all game screens
4. Use character for motivational messages

## Free Character Resources

You can find free character illustrations at:
- https://www.flaticon.com/ (character illustrations)
- https://www.freepik.com/ (character designs)
- https://undraw.co/illustrations (customizable illustrations)
- https://www.humaaans.com/ (mix-and-match characters)

Or create custom characters using:
- Figma
- Adobe Illustrator
- Canva
- Procreate

## Note

If character images are not present, the system will display a fallback icon. The app will function normally, but the character experience will be limited.
