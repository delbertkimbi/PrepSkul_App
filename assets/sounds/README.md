# Game Sound Effects

This directory contains sound effect files for the skulMate games.

## Required Sound Files

Add the following sound files to this directory:

1. **correct.mp3** - Played when user answers correctly
2. **incorrect.mp3** - Played when user answers incorrectly
3. **flip.mp3** - Played when flipping flashcards
4. **match.mp3** - Played when making a correct match in matching game
5. **complete.mp3** - Played when game is completed
6. **click.mp3** - Played on button clicks (optional)

## Sound File Recommendations

- **Format**: MP3 (recommended for cross-platform compatibility)
- **Duration**: 0.5-2 seconds
- **Volume**: Normalized to avoid sudden loud sounds
- **License**: Ensure you have proper licensing for any sound files used
