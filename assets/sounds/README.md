# Game Sound Effects

Add these MP3 files for full game audio. Place them in this folder:

| File | Use |
|------|-----|
| `flip.mp3` | Card flip, swipe |
| `correct.mp3` | Right answer |
| `incorrect.mp3` | Wrong answer, no match |
| `match.mp3` | Matching pair found |
| `complete.mp3` | Game completed |
| `click.mp3` | Button tap |
| `pop.mp3` | Bubble pop |
| `wordFound.mp3` | Word search hit |
| `piecePlace.mp3` | Puzzle piece placed |

**Free sources:** [Freesound.org](https://freesound.org), [Mixkit](https://mixkit.co/free-sound-effects/game/), [Pixabay](https://pixabay.com/sound-effects/search/game/).

**If files are missing:** The app falls back to system haptic feedback and click sounds.
