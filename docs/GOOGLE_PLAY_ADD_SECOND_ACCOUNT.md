# How to Add a Second Account in Google Play Console

## 📋 You Have One Entry - Here's How to Add the Second

## 🔍 Step-by-Step: Adding Second Account

### **Method 1: After Saving First Account**

1. **After you save the tutor account**, you should see:
   - Your saved tutor account listed
   - A button that says: **"Add another instruction"** or **"Add instruction"** or **"+"** button

2. **Click "Add another instruction"** (or similar button)

3. **Fill in the student account form:**
   - **Instruction name:** `Student Account - PrepSkul Demo`
   - **Username/Email:** `prepskulcode@gmail.com`
   - **Password:** `DE12kimb`
   - **Any other information:** `Demo Student Account - Pre-approved with full access. All student features available: dashboard, tutor search, bookings, sessions, payments. Contact support@prepskul.com if needed.`

4. **Click "Save"**

---

### **Method 2: If You Don't See "Add Another" Button**

1. **Go back to the App access page:**
   - Navigate to: **Policy** → **App content** → **App access**

2. **Look for your saved tutor account:**
   - You should see it listed

3. **Look for options:**
   - **"Add instruction"** button
   - **"+"** icon
   - **"Add another"** link
   - Or an **"Edit"** button that might let you add more

4. **Click to add a new instruction**

5. **Fill in the student account details** (same as Method 1)

---

### **Method 3: Edit Mode**

1. **If you're still in edit mode** for the first account:
   - Look for a button like **"Add instruction"** or **"Add another"** 
   - It might be at the top or bottom of the form
   - Click it to add a second entry

2. **Fill in both accounts:**
   - First entry: Tutor account
   - Second entry: Student account

3. **Save both at once**

## 🎯 Alternative: One Account is Enough

**Note:** If you can't find how to add a second account, **one account (tutor) is usually sufficient** for Google Play review. They mainly need to verify the app works, and the tutor account gives access to all features.

However, if your app has significantly different features for students vs tutors, having both is helpful.

---

## 🆘 Still Can't Find It?

If you absolutely cannot find how to add a second account:

1. **Check if there's a limit:**
   - Some Play Console sections only allow one entry
   - In that case, use the **tutor account** (it has more features)

2. **Contact Google Play Support:**
   - They can help you add multiple accounts if needed

3. **Use just the tutor account:**
   - It's pre-approved and has full access
   - Reviewers can test all app features with it
