## ⚠️ Important Notes

### **Before You Save:**

- ✅ Make sure you've created the demo account in your app first
- ✅ Make sure the demo account is approved in your admin dashboard
- ✅ Test the credentials yourself to ensure they work
- ✅ Double-check the email and password are correct

### **If You Don't Have a Demo Account Yet:**

1. **Create one first:**
   - Open your PrepSkul app
   - Sign up as a tutor with email: `demo@prepskul.com`
   - Complete onboarding
   - Approve it in admin dashboard: `https://admin.prepskul.com/admin/tutors`

2. **Then come back** to this page and add the credentials

## ❓ Troubleshooting

**Q: I don't see the credential fields after selecting "restricted"?**
- A: Try refreshing the page or clicking "Save" first, then edit again

**Q: What if I don't have a demo account yet?**
- A: Create one first (see instructions above), then come back to add credentials

**Q: Can I add multiple accounts?**
- A: Usually one account is enough, but you can add multiple if needed

**Q: What if I make a mistake?**
- A: You can always come back to this page and edit the credentials
