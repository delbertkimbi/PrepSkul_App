# PrepSkul Virtual Assistant vs PrepSkul AI - Naming Recommendation

## Decision: **PrepSkul Virtual Assistant** (or "PrepSkul VA")

### Why "Virtual Assistant" Over "AI"?

#### 1. **Trust-Building with Parents**
- "Virtual Assistant" sounds more **human-like** and **supportive**
- "AI" can sound **cold, automated, or impersonal**
- Parents want to feel their child has **human support**, not just a robot
- "Assistant" implies **help and guidance**, which aligns with tutoring values

#### 2. **Professional Image for Investors**
- "Virtual Assistant" sounds more **mature and established**
- "AI" might sound like **experimental technology**
- Investors prefer **proven, professional solutions**
- "Assistant" suggests **reliability and support**

#### 3. **Comfort for Tutors**
- Tutors might feel threatened by "AI" (replacement fear)
- "Virtual Assistant" sounds like a **tool to help them**, not replace them
- Emphasizes **collaboration** rather than automation
- More welcoming for tutors who aren't tech-savvy

#### 4. **Marketing & Communication**
- Easier to explain: "Our virtual assistant joins sessions to help with notes"
- "AI" requires more explanation and might confuse non-tech users
- "VA" is a friendly, approachable abbreviation
- Better for customer support conversations

#### 5. **Brand Consistency**
- Aligns with PrepSkul's mission: **supporting learners**
- "Assistant" reinforces the **helping, supportive** brand
- Less intimidating than "AI"
- More accessible to all user types

### When to Mention Fathom

**Don't mention Fathom unless:**
- Asked specifically about the technology
- Discussing technical implementation with developers
- Required for compliance/transparency documentation

**Instead, say:**
- "PrepSkul Virtual Assistant"
- "Our virtual assistant"
- "PrepSkul VA"
- "The assistant that joins sessions"
